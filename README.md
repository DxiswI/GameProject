# Game Project

## Описание
Это проект игры, написанный на Java.

## Задание
- Создать репозиторий и закомитить ветку main
- Создать ветку omniwin где поправить код так чтобы в игре никто не проигрывал
- Создать ветку omnilose где поправить код так что в игре все проигрывают


## Структура проекта
- src/main/java/org/example/Main.java - основной код игры
- pom.xml - конфигурация Maven
- .gitignore - файл для игнорирования ненужных файлов
