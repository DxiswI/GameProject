# Game Project

## Описание
Это проект игры, написанный на Java.

## Задание
1.Создать репозиторий и закомитить ветку main
2.Создать ветку omniwin где поправить код так чтобы в игре никто не проигрывал
3.Создать ветку omnilose где поправить код так что в игре все проигрывают


## Структура проекта
- src/main/java/org/example/Main.java - основной код игры
- pom.xml - конфигурация Maven
- .gitignore - файл для игнорирования ненужных файлов
